- Developed and deployed a full-stack second brain application using Next.js 16 (TypeScript), FastAPI, and Supabase, supporting secure capture and fast retrieval of notes, links, and code snippets with search, tag filters, and pin/archive workflows.

- Applied production-ready standards with JWKS-based JWT authentication, per-user data isolation, consistent API error handling, rate limiting, accessibility-focused UI improvements, and automated GitHub Actions pipelines for dependable releases.
